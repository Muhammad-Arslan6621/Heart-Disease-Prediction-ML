# Heart Disease Risk Prediction using Machine Learning

This project focuses on classifying heart disease risk levels using public health data. By implementing a Random Forest Classifier, the model achieves a high accuracy rate, making it a reliable tool for preliminary risk assessment.

## Key Features
* **Data Cleaning:** Handled over 600,000 rows of data, managing mixed units and missing values.
* **Model:** Used **Random Forest Classifier** for robust predictions.
* **Accuracy:** Achieved an impressive **96.26%** on the test set.
* **Tuning:** Performed Hyperparameter tuning using `RandomizedSearchCV`.

## Technical Stack
* **Language:** Python
* **Libraries:** Scikit-Learn, Pandas, NumPy, Matplotlib
* **Environment:** Google Colab
